# マルチログインECサイト

## 動作環境
* Ubuntu 22.04
* PHP 8.1.20
* Laravel Framework 8.83.27
* MySQL 8.0
* Docker version 24.0.2
* phpMyAdmin 5.2.1 [データベースの中身はこちらから](http://116.80.77.78:4040/)
